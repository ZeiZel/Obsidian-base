# Инструменты

Раздел сгруппирован по задачам, чтобы быстрее выбирать инструмент под конкретный слой разработки.

## validators

- [[validators/Zod|Zod]] - универсальная runtime-валидация с сильной TypeScript-интеграцией.
- [[validators/Valibot|Valibot]] - модульная и лёгкая валидация для client/edge-бандлов.
- [[validators/TypeBox|TypeBox]] - TypeScript-типы и JSON Schema из одной схемы.
- [[validators/Ajv|Ajv]] - быстрый валидатор JSON Schema.

## engines

- [[engines/Bun|Bun]] - runtime, package manager, test runner и bundler в одном инструменте.
- [[engines/tsx|tsx]] - запуск TypeScript-файлов в Node.js без ручной компиляции.

## build-tools

- [[build-tools/ESBuild|ESBuild]] - быстрый bundler/transpiler.
- [[build-tools/Webpack|Webpack]] - гибкий сборщик для сложных и legacy-проектов.
- [[build-tools/tsup|tsup]] - сборка TypeScript-библиотек, SDK и CLI.

## package-managers

- [[package-managers/PNPM|PNPM]] - быстрый пакетный менеджер с сильной поддержкой workspaces.
- [[package-managers/NCU|NCU]] - проверка и обновление версий зависимостей.

## quality-tools

- [[quality-tools/ESLint|ESLint]] - статический анализ и правила качества кода.
- [[quality-tools/Prettier|Prettier]] - форматирование кода.
- [[quality-tools/TypeScript Config|TypeScript Config]] - строгие настройки `tsconfig.json`.
- [[quality-tools/ts-reset|ts-reset]] - более безопасные стандартные TypeScript-типы.

## testing

- [[testing/Vitest|Vitest]] - быстрый test runner для TypeScript/JavaScript.

## monorepo

- [[monorepo/NX|NX]] - управление monorepo, graph, cache и affected-команды.

## editor

- [[editor/VSCode|VSCode]] - настройки редактора и расширения для TypeScript-разработки.

## utilities

- [[utilities/ts-pattern|ts-pattern]] - pattern matching и exhaustive-проверки union-типов.
- [[utilities/neverthrow|neverthrow]] - явная обработка ошибок через `Result`.
