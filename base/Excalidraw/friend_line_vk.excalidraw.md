Формулы

1. Нагрузка:
RPS (write) = 50_000_000 (dau) / 5 (days) / 86400 ~= 115 rps
RPS (read) = 50_000_000 * 10 / 86400 = 5787 rps ~= 5800 rps

ACR - Architecture Cignificent Requirements - ключевые требования, которые нужно соблюсти. 
Без них система будет бесполезна и не будет эффективно выполнять задачи. 

Так как на чтение нагрузка почти в 50 раз выше, то поэтому обязательно нужно учесть, что система Read-intensive. 

2. Traffic:

Write media = 115 * 1MB = 115MB/s
Meta info = 115 * 2000 (symbols) * 2byte (symbol/utf-8) = 115 * 4KB = 460 kb/s
Read media = 5800 (read) * 10 (постов отдаём пользователю за раз) * 1MB (вес изображения) = 58GB/s
Read meta = 5800 * 10 * 4KB = 232MB/s
